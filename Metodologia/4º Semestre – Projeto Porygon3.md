## 4º Semestre – Projeto Porygon3

Empresa parceira: GSW

O projeto Porygon3 teve como objetivo a evolução do sistema das versões anteriores, incorporando novos módulos, melhorias de performance, dashboards, relatórios e aperfeiçoamento da arquitetura. A solução desenvolvida foca em entregar uma plataforma mais robusta, eficiente e escalável, alinhada às necessidades da empresa parceira.

Repositório GitHub

Tecnologias Utilizadas

Java + Spring Boot – Evolução da API com arquiteturas mais profissionais.

MySQL / JPA – Modelo de dados ampliado e otimizado.

Vue.js – Interface moderna e reativa.

Axios – Comunicação com backend.

Leaflet – Exibição de mapas e dados geoespaciais.

Git / GitHub Flow – Processo de versionamento maduro.

Contribuições Pessoais

Evoluí módulos já existentes, corrigindo bugs, criando novas validações e garantindo integridade nos dados.

Implementei endpoints avançados no backend, utilizando filtros dinâmicos, paginação e consultas especializadas.

Trabalhei diretamente com o frontend em Vue.js para construir telas mais intuitivas e responsivas.

Realizei a integração backend + frontend, garantindo consistência entre JSONs, DTOs e modelos de tela.

Participei da refatoração de parte da arquitetura, tornando o código mais limpo, padronizado e modular.

Ajustei o fluxo de CI/CD dentro do repositório, contribuindo para builds mais confiáveis durante as sprints.

Hard Skills Desenvolvidas

Vue.js – construção de interfaces modernas e integrações REST.

Spring Boot – desenvolvimento intermediário/avançado de APIs.

Integração de sistemas – domínio maior de comunicação entre camadas.

Mapas com Leaflet – construção de componentes geoespaciais.

Soft Skills Desenvolvidas

Resolução de problemas complexos: bugs envolvendo dados e regras de negócio.

Comunicação técnica: justificativa de soluções e decisões arquiteturais.

Colaboração: forte interação com a equipe para garantir qualidade e consistência.

Liderança técnica informal: apoio aos colegas em dificuldades com backend.
